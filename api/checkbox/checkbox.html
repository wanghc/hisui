选择框，源来Bootstrap的iCheck插件。

说明:

点击勾选，常用于多项选择。

如：

<input class='hisui-checkbox' type="checkbox" data-options="onCheckChange:function(event,value){console.log(value)}" label="已出院" id="checkbox1">
<input class='hisui-checkbox' type="checkbox" checked label="已停止">
<input class='hisui-checkbox' type="checkbox" data-options="checked:true" label="已帐单">
<input class='hisui-checkbox' type="checkbox" label="已就诊">

复选框组件的相关配置、事件、方法

属性 说明 默认值
id 复选框的Id ''
label 描述 ''
disabled 是否禁用。默认可用 false
checked 是否选中。默认不选中 false
事件名 说明 入参
onCheckChange 选中改变事件 e,value
onChecked 选中事件 e,value
onUnchecked 取消选中时触发 e,value
方法名 说明 入参 返回值
options 拿到配置项对象 checkbox的options对象
setValue 选中元素 true|false jquery对象
getValue 拿到是否选中 true|false
setDisable 禁用 false|true jquery对象
check 选中 jquery对象
uncheck 取消选中 jquery对象
toggle 切换选中 jquery对象
disable 禁用选框 jquery对象
enable 启用选框 jquery对象
indeterminate 设置不定选状态 jquery对象
determinate 取消不定选状态 jquery对象
update 重绘选框 jquery对象
destroy 销毁选框 jquery对象
