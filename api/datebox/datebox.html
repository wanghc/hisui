日期框

说明:

日期框（datebox）把可编辑的文本框和下拉日历面板结合起来，用户可以从下拉日历面板中选择日期，在文本框中输入的字符串可转换为有效日期。

如：

生日:
Selected Date: 
<table cellpadding="5"> 
	<tr>
		<td>生日:</td>
		<td>
			<input id="db" class="hisui-datebox textbox" data-options="onSelect:onSelect"></input>
		</td>
	</tr>
</table>
<div style="margin:10px 0">
	<span>Selected Date: </span>
	<span id="result"></span>
</div>
<script type="text/javascript">
function onSelect(date){
	$('#result').text(date)
}
</script>
