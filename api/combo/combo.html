组合框

说明:

组合（combo）是在 html 页面上显示一个可编辑的文本框和下拉面板。它是用于创建其他复杂的组合组件（比如：组合框 combobox、组合树 combotree、组合网格 combogrid，等等）的基础组件。

组合（combo）可以使用 javascript 从 <select> 或 <input> 元素进行创建。请注意，从标记创建组合（combo）是无效的。

如：

<input id="cc" placeholder="空面板">
<script type="text/javascript">
$(function(){
	var ccObj=$HUI.combo('#cc',{});
});
</script>

$(function(){
	var cc2Obj=$HUI.combo('#cc2',{editable:false,placeholder:'输入查询条件'});
	$('#sp').appendTo(cc2Obj.panel());
	$('#sp input').radio({onCheckChange:function(e,v){
		var v = $(this).val();
		var l = $(this).attr('label');
		cc2Obj.setValue(v);
		cc2Obj.setText(l);
		cc2Obj.hidePanel();
	}});
});
